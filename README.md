# Verdant
Verdant is 40% keyboard with OLED and rotary encoder
- Uses Pro Micro 
- Pcb gasket mount
- O-ring gaskets
- Stacked acrylic case
- Rgb underglow

Changes from Verdant 0.1 to 0.2

- More layout support (split space, ansi enter, 2U Lshift)
- Plate
- Smd rgb leds instead of strip
- Acrylic legs
